# Faster-Minecarts-bedrock
Faster Minecarts

Basically makes minecarts go faster.


## Installation

Install my project by going into the [Latest Release](https://github.com/CoderandHacker69/Faster-Minecarts-bedrock/releases)
## Usage/Examples
Normal usage: Sneak and Interact with Minecart. (Expirimental)

Command Usage:
```mcfunction
/scoreboard players set @p speed_level [lvl 1-5]
```
Then Run:
```mcfunction
/function mine
```
